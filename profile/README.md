# 👋 Welcome to LoomForge

LoomForge is a community dedicated to crafting high-quality software — weaving together ideas, tools, and contributors to build things that matter.

## 🧵 What We Build

We create open-source projects focused on developer tooling, automation, and infrastructure. Our work spans a range of domains, but we share a common commitment to clean design, solid engineering, and collaborative development.

## 🤝 Get Involved

We welcome contributions from developers of all backgrounds and experience levels. Here's how you can get involved:

- **Explore our repositories** — browse our projects and find something that interests you
- **Open an issue** — report a bug, request a feature, or start a discussion
- **Submit a pull request** — contribute code, docs, or tests to any of our projects
- **Join the conversation** — share ideas and feedback with the community

## 📋 Contributing

Each repository has its own `CONTRIBUTING.md` with project-specific guidelines. In general, we ask that contributors:

- Follow the existing code style and conventions
- Write clear commit messages
- Add tests for new functionality
- Be respectful and constructive in all interactions
